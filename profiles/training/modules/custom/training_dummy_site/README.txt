-- SUMMARY --

It helps in creating the dummy site for training website.

It provides following features :
1. It sets the Calendar View as the home page.
2. It generates dummy data related to built in content type in training 
installation.

To configure this module :
1. Navigate to 'admin/config/people/training-dummy-site'.
2. Choose the setting you want for your dummy site.

-- INSTALLATION --

See the installation guide here - http://drupal.org/node/70151.
